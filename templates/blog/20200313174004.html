    Apache access log and error log locations The location of Apache access logs and error logs on Ubuntu is defined by the environment variable "APACHE_LOG_DIR". It is defined in the "envvars" file in the directory where the Apache configuration files are located . 
    export APACHE_LOG_DIR = / var / log / apache2 $ SUFFIX


$ SUFFIX is an empty string when only one Apache is running, so the Apache log directory will be "/ var / log / apache2".


    sudo ls / var / log / apache2
