postgresqlのインストール - PostgreSQLサーバー

PostgreSQLデータベースサーバーを使えるように「postgresql」をインストールしましょう。PostgreSQLは、利便性の高いリレーショナルデータベースシステムです。 
sudo apt install -y postgresql

PostgreSQLデータベースサーバーがインストールされたことの確認

PostgreSQLデータベースサーバーがインストールされたことをsystemctlコマンドのstatusを使って、確認しましょう。 
systemctl status postgresql
次のように状態が表示されます。
