Show a list of correspondence between used port numbers and servers --lsof -i TCP -P

To see a list of correspondences between used port numbers and servers, use the lsof command and specify the following options: 
lsof -i TCP -P
The "-i" option is limited to TCP only. Practically, if you can see only TCP, there is no duplication and it is easy to see. With the "-P" option, the port is displayed without displaying the protocol (http, etc.). This is a sample output result of lsof.
