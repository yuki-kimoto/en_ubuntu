Installing vsftpd-FTP server

To use the FTP server, install vsftpd with the apt command. 
sudo apt install -y vsftpd
FTP stands for File Transfer Protocol. A protocol for exchanging files on a client / server. FTP is not encrypted. Use SFTP built on SSH to transfer confidential information that requires encryption.

Confirm that vsftpd is installed

Use the status of the systemctl command to verify that vsftpd is installed. 
systemctl status vsftpd
