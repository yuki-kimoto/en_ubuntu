sshパッケージのインストール - SSHサーバーとクライアント

SSHサーバーとSSLクライアントを使えるようにするにはsshパッケージをインストールします。 
# SSHサーバーとSSHクライアントのインストール
sudo apt install ssh
これはOpenSSHと呼ばれるSSHクライアントとSSHサーバーを含むオープンソースのプロジェクトです。 SSHサーバーとクライアントが、デフォルトでインストールされていない場合に、インストールしてください。 SSHサーバー、SSHクライアントを個別にインストールすることもできます。 
# SSHクライアント
sudo apt install ssh openssh-client

# SSHサーバー
sudo apt install ssh openssh-server

sshコマンドがインストールされたことの確認

sshコマンドがインストールされたことを確認しましょう。 
ssh --help

sshサーバーがインストールされたことの確認

sshサーバーがインストールされたことをsystemctlコマンドで確認しましょう。sudoで管理者権限で実行します。 
sudo systemctl status ssh
activeと表示されれば、インストールが成功し、SSHサーバーが起動しています。
