Installing mariadb-client --MariaDB database client

MariaDB database client application "mysql" can be used with "mariadb-client" apt Let's install it with the command. 
sudo apt install -y mariadb-client

Confirm that the mysql command is installed

Make sure you have MariaDB's mysql command installed. 
mysql --version
It is installed when the version information is displayed as shown below. 
mysql Ver 15.1 Distrib 10.1.44-MariaDB, for debian-linux-gnu (x86_64) using readline 5.2
