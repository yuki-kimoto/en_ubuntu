    Installing libopencv-dev --OpenCV library header files and libraries I will explain how to install libopencv-dev. Install libopencv-dev using the apt command. Run with administrator privileges using sudo. 
    sudo apt install -y libopencv-dev


For the location of the installed OpenCV header files and libraries, see  Header file read path  and  Library loading path .
