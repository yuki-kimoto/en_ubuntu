    Installing the libio-socket-ssl-perl package-Perl IO :: Socket :: SSL Install the libio-socket-ssl-perl package with the apt command for HTTPS access in Perl. Run with administrator privileges using sudo. 
    sudo apt install -y libio-socket-ssl-perl


Perl's IO :: Socket :: SSL module is installed. Net :: SSLeay also depends on IO :: Socket :: SSL, so it will be installed.
