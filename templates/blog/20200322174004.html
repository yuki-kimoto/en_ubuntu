    How Apache child process execution user writes to data directory In Ubuntu Apache, after starting with root privileges, the child process that actually receives the request is executed by the user "www-data". Let's take a look at the actual process with the ps command. Use the grep command to narrow down the apache. 
    ps -ef | grep apache
