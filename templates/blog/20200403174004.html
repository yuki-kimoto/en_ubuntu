"/ etc / group" file showing all groups and users belonging to the group

The list of groups can be found in the "/ etc / group" file. Use the cat command to display the "/ etc / group" file. Execute it with root privileges using the sudo command. You can also use "/ etc / group" to find out who belongs to a group. 
sudo cat / etc / group
