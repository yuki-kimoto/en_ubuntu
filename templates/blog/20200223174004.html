Ubuntu MariaDB server configuration file location

The location of the MariaDB configuration file on Ubuntu is in the following directory: 
/etc/mysql/mariadb.conf.d
There are four configuration files in this. The MariaDB server configuration file is "50-server.cnf". 
50-client.cnf 50-mysql-clients.cnf 50-mysqld_safe.cnf 50-server.cnf
