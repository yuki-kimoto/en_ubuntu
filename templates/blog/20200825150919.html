vimのインストール

vimのインストール方法を解説します。aptコマンドを使って、vimをインストールします。sudoを使って管理者権限で実行します。 
sudo apt install -y vim

Ubuntuにおけるvimとviの違い

Ubuntuにおいてはデフォルトでは、viには、vimの機能を限定したエディタvim-tinyが割り当てられています。 viの編集モードで、上下左右のキーが使えず「A B C D」のように表示されてしまうのは、viがvim-tinyであることが原因です。 このような場合は、上記の手順でvimをインストールすれば、viにvimが割り当てられます。 (Ubuntu 18で確認)
