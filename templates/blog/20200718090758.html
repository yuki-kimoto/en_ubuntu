    Enable the proxy_wstunnel module for WebSocket reverse proxy in Apache If you have a WebSocket web application, enable the proxy_wstunnel module to reverse proxy with Apache. 
    sudo a2enmod proxy_wstunnel


Please  Restart Apache  for the settings to take effect.

Please refer to the following articles for how to enable / disable Apache modules.
