Ubuntu version check

To check the Ubuntu version, use the cat command to "/ etc / os-release" Outputs the contents of. 
cat / etc / os-release
Version output example. 
NAME = "Ubuntu"
VERSION = "18.04.4 LTS (Bionic Beaver)"
ID = ubuntu
ID_LIKE = debian
