Enable mod_proxy module in Apache

In Ubuntu Apache, the mod_proxy module that provides proxy functionality is not enabled by default, so enable it. 
sudo a2enmod proxy
Please Restart Apache for the settings to take effect. Please refer to the following articles for how to enable / disable Apache modules.
  • Enable / disable Apache config file-basic config / module config / site config
Please note that mod_proxy_http is required separately to proxy HTTP / HTTPS.
