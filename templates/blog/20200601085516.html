Check the physically attached disk --fdisk -l

Use "fdisk -l" to see which disks are physically attached to Ubuntu. It is assumed that an external hard disk is connected or a hard disk is added (attached) from the GUI screen using a cloud service. This state is not yet recognized by the Linux file system, and a device called a hard disk is physically connected. First, use "fdisk -l" to check the status of the connected disk. Run with administrator privileges using sudo. 
sudo fdisk -l
