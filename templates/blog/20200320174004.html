Apacheにmod_phpをインストールする

Apacheにmod_phpをインストールします。Apacheで、PHPアプリケーション動かす必要がある場合に、必要です。 
sudo apt install libapache2-mod-php
mod_phpをインストールしたら、次は、mod_phpを有効にしましょう。(2020年6月8日記載。Ubuntu 18。PHPバージョンは7.2) 
sudo a2enmod php7.2
最後にApacheを再起動して完了です。 
sudo systemctl reload apache2
これで、PHPアプリケーションがApache上で動くようになりました。
