How to install XML :: LibXML with cpanm in user Perl.

XML :: Package required for LibXML

The libxml2-dev package is required to install XML :: LibXML.
  • Installing libxml2-dev --Libxml2 headers and libraries

XML :: Install LibXML 

cpanm XML :: LibXML
Make sure you can read XML :: LibXML. 
perl -e'use XML :: LibXML'
