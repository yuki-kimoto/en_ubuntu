    Enable mod_proxy_http module in Apache In Apache of Ubuntu, the mod_proxy_http module that provides the proxy function by HTTP / HTTPS is not enabled by default, so enable it. 
    sudo a2enmod proxy_http


Please  Restart Apache  for the settings to take effect.

Please refer to the following articles for how to enable / disable Apache modules.


      
  
    •   Enable / disable Apache config file-basic config / module config / site config  


Note that mod_proxy_http depends on mod_proxy.
