mariadb-clientのインストール - MariaDBデータベースクライアント

MariaDBデータベースのクライアントアプリケーションである「mysql」を使えるように「mariadb-client」をインストールしましょう。 
sudo apt install mariadb-client

mysqlコマンドがインストールされたことの確認

MariaDBのmysqlコマンドがインストールされたことを確認しましょう。 
mysql --version
以下のようにバージョン情報が表示されるとインストールされています。 
mysql  Ver 15.1 Distrib 10.1.44-MariaDB, for debian-linux-gnu (x86_64) using readline 5.2

DBD::MySQLを使える用にするには

(2020年5月18日時点で、確認未) PerlのDBD::MySQLを使るようにするには「libmariadb-dev」パッケージが必要になります。
